# git-checkup

Git 规范检测工具

## Features

关于分支
- [ ] 分支命名规范
- [ ] 分支合并提醒
- [ ] 分支自动合并，支持定时
- [ ] 分支异常推送

关于tag
- [ ] 批量生成分支，支持定时
- [ ] tag 命名规范
- [ ] tag 检测与提醒（发版半日后）
- [ ] 批量生成tag，支持定时

关于提交
- [ ] commit meassage检测
- [ ] commit 提醒，每天至少一次

关于代码
- [ ] 代码注释检测
- [ ] 代码规范检测
- [ ] 新代码review提醒

关于仓库
- [ ] 仓库命名规范

关于user
- [ ] user账号信息检测，异常提醒
 

## configuration

- 需要检测的项目列表
- 项目迭代开始结束时间的数组
- 检测时间节点
- 分支、tag、提交信息等规范的正则
- ...
